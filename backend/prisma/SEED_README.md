## 4. Other Details

- **Images**: Products and Outlets are seeded with Unsplash/Picsum images.
- **Stock**: Initial stock for GOODS is randomized (10-60 units).
- **Operating Hours**: All outlets operate Mon-Sat (09:00 - 21:00 WIB / 02:00 - 14:00 UTC). Closed on Sundays.

## 5. Usage

To re-run the seed and **reset the database** (WARNING: Deletes all data):

```bash
npx prisma db seed -- --force
# OR
npm run seed -- --force
```
