## 5. FNB Transactions (100 Sample Orders)

The FNB outlet (`Kopi Nusantara - Kemang`) has **100 seeded transactions** using "Kopi Susu Spesial" with PPN 11%.

- Each order has 1-4 items with qty 1-3.
- `taxAmount` = `Σ Math.round(price × qty × 11 / 100)` — matches service-layer logic.
- `totalAmount` = `subtotal + taxAmount` (fees stored separately).
- `handledByStaffId` = `Kasir FNB` for all 100 orders.
- Guest customers are randomized (Indonesian names via faker).
- Payment methods randomized: qris, gopay, bca_va, mandiri_va, cash.
- Dates spread across the last 6 months.
- Stock decrements and stock logs are created per transaction.

## 8. Tax Calculation (Service-Layer Aligned)

All tax calculations follow the actual service logic found in `order-create.helper.ts`:

```typescript
// Per item:
totalTax += Math.round(sellingPrice * quantity * (taxPercentage / 100));

// Order totals:
totalAmount = subtotal + totalTax
// midtransFee and appFee are stored separately on Order (not included in totalAmount)
```

## 9. Usage

The seed **always clears all data and re-seeds from scratch** — there is no `--force` flag check.

```bash
npx prisma db seed
# OR
npm run db:seed
```
